# App

Le client (front-end) du projet NoDesk de gestion de "pré"-dossiers et documents
rapides.

# Framework

* angularJS
* ionic
* UI Bootstrap
* Cordova/PhoneGap

# Développement

## Browser

* Serveur apache
* Cloner dans www/ (ou html_public)
* Tester dans chrome avec le plugin ripple (http://emulate.phonegap.com/)

## Emulator

* Packager avec PhoneGap Build (https://build.phonegap.com/)
* Android SDK
* ios emulator (XCode)
